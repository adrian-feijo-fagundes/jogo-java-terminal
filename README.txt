COMO O JOGO VAI FUNCIONAR?


AO INICIAR:

- Qual o nome do seu personagem?

- Escolhar uma classe para seu personagem:
    - Mercador
    - Aventureiro
    - Sacerdote
    - Guerreiro

- Quais os atributos dos personagens?
    - Vida          HP
    - Mana          MP
    - Força         ATK
    - Defesa        DEF
        - Reduz o dano recebido
    - Velocidade    SPD
        - Prioridade ao atacar
        - Chance de esquiva
    
    - Sorte         LUK
        - Chance de Critico
        - Chance de Fuga
        - Chance de itens raros
        - Chance de eventos aleatórios favoráveis


    O personagem tambem possui:
        Dinheiro
        Equipamentos para:
            Cabeça
            Peitoral
            Calças
            Calcados
            Acessório
            Arma principal
            Arma Secundária

- O seu personagem pode:
    Caminhar
    Equipar itens
    Usar itens

- Sistema de Combate
    durante o combate:
        atacar
        usar habilidade
        usar item
        fugir

- Sistema de escolhas e estrutura da história
    O jogador pode escolher entre: DIREITA e ESQUERDA

- Alguns eventos que podem acontecer:
    Encontrar inimigo
    Encontrar tesouro
    Encontrar Cambista 
    Encontrar saída do labirinto
